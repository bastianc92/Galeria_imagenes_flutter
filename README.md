# Galeria_imagenes_flutter

Aplicación móvil desarrollada en Flutter que permite visualizar, recortar, compartir y almacenar imágenes usando Firebase Storage.

## Funcionalidades
- Visualización de imágenes locales y en la nube
- Recorte de imágenes con `image_cropper`
- Compartir imágenes con `share_plus`
- Zoom y vista completa con `photo_view`
- Subida y descarga desde Firebase Storage

## Requisitos
- Flutter 3.x
- Android SDK
- Firebase configurado con `google-services.json`
